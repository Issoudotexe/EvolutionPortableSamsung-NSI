Les outils de base :
* un éditeur de texte suuportant HTML, CSS, javascript avec la coloration syntaxique : notepad++ par exemple
* un navigateur web : firefox par exemple

vocabulaire à maitriser : éditeur de texte, navigateur, coloration syntaxique

Gestion des fichiers :
* créer un dossier pour vos projets si pas déjà fait
* créer un sous-dossier site_test
* créer un fichier index.html, un dossier images, un dossier styles, un
dossier scripts
* comprendre les chemins d'accès : parent ../

vocabulaire à maitriser : dossier, sous-dossier, répertoire, sous répertoire, extension, répertoire parent, chemin d'accès 

Le langage HTML : hyper text markup langage

il sert à définir le contenu, c'est un langage de description (pas de boucle)

vocabulaire à maitriser : balise ouvrante, fermante, élément, attribut
comprendre l'imbrication des éléments
éléments à connaitre : html, head, body,
                        title (ne pas confondre avec h1) , img
                        paragraphe, liste ordonnée, liste non ordonnée
                        liens a et link (feuille de style)

attributs à connaitre : id, class (feuille de style)

Le langage CSS : cascading stylesheets

il sert à définir la mise en page, c'est un langage de description (pas
de boucle)

vocabulaire à maitriser : feuille de style, sélecteur, propriété
sélecteurs à connaitre : élément html, #id, .class, attribut
(element[attribut])

Le langage javascript :

il sert à gérer des événements côté client

vocabulaire à maitriser : script, événement
